# Hamilton & Forge Digital – 7-Day Action Plan

This is a simple, interactive one-page website designed to guide a one-week digital presence rollout.

## Features
- Visual daily timeline
- Actionable checklists
- Progress saved in browser
- One-click print-ready layout
- GitHub Pages compatible

## Deployment
1. Push this repo to GitHub
2. Enable **Settings → Pages**
3. Set source to `main` / root
4. Access via: https://<username>.github.io/<repo>/
